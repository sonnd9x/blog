## Hướng dẫn

### Các tính năng
- Compatible with GitHub Pages
- Responsive design (looks just as good on mobile)
- Syntax highlighting, with the help of Pygments
- Markdown and HTML text formatting
- Pagination of posts
- Sticky posts
- Tags
- Excerpt management
- Disqus comments

**Ví dụ bài viết**

```
---
layout: post
title:  "Introducing Tale"
author: "Chester"
comments: true
tags: Tale
excerpt_separator: <!--more-->
sticky: true
hidden: true
---
```

**Ví dụ trang**
```
---
layout: post
title: "About"
author: "Chester"
permalink: /about/
---
```
